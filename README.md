## Overview of CS2 Stats for Steam Profiles

CS2 Stats for Steam Profiles is a Chrome extension that adds Counter-Strike 2 context directly to Steam profile pages. It is aimed at users who want competitive information without jumping between tabs or separate services.

The extension pulls together data from Steam, Leetify, and Faceit and presents it in a panel on the right side of the profile. It is built for Manifest V3 and keeps setup lightweight while still showing details like XP, rank, friend code, commendations, match history, and KD calculations.

## What it includes

- Displays CS2 stats while viewing Steam profiles
- Brings Steam, Leetify, and Faceit data into one panel
- Uses a right-aligned profile UI for fast scanning
- Shows XP, rank, friend code, and commendation information
- Includes match history and KD calculation
- Built as a Manifest V3 Chrome extension
- Supports Faceit API key usage through a proxy flow
